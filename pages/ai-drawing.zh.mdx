# AI 画图

## 快速上手
```
开始画图时，请确保你的App版本高于1.0.10，桌面端版本高于1.0.7；
```
请在Hayo首页，选择“AI画图”进入画图界面；此时在输入框内，输入想要绘制的画面，提交后，AI将开始处理命令，如果你的命令正确无误，那么AI将开始作图，作图时间在20s左右。
你也可以从一个画图指令开始，在首页-灵感库-AI画图分类下，查看热门的画图指令，点击“画图”即可使用该指令画图。

## 常用命令
在你的画图工作台里，输入“/”，或者直接点击“/”会激活命令界面，此时你会看到画图机器人的全部命令：
- ``/画图``使用AI画图；
- ``/帮助``查看画图机器人的帮助文档。

## 画图参数

### --参数的定义
画图时，在你的关键词结尾添加的控制参数，结构是``--参数 <参数值>``。
```
注意添加参数时，关键词和参数之间需要严格按照格式：关键词 --参数 --参数，否则会不生效
```

### --参数的特性
- 单次作图，可设置多参数；
- 同一个参数不能重复，比如不能设置两个--模型；

### --参数列表
#### --模型
用于切换底模，这些模型已经支持简中、繁中，当前已经支持的模型有：

| --模型英文 | --模型简中 | --模型繁中 | 模型介绍 |
| ----- | ---- | ---- | ---- |
| --real (默认) | --照片 | --照片 | 可以较好生成真实照片，比如动植物、室内设计、风景等；|
| --beauty | --人像 | --人像 | 可以较好生成人像照片；|
| --anime | --动漫 | --動漫 | 可以较好生成二次元动漫风格图；|
| --cute | --可爱 | --可愛 | 可以生成皮克斯风格的图片，对于相关人物的名称兼容度不错，能识别类似Elsa、Cinderella等；|
| --dream | --艺术 | --藝術 | 可以生成艺术化的图片，如梦境般唯美； |

#### --ar 宽:高
用于更换图片尺寸

| ``--ar <宽>:<高>`` | 对应图片比例 | 分辨率 |
| ----- | ---- | ---- |
| --ar 1:1 （默认） | 1:1 | 768*768 |
| --ar 16:9 | 16:9（1.77） | 910*512 |
| --ar 9:16 | 9:16 | 512*910 |
| --ar 3:4 | 3:4 | 512*681 |
| --ar 4:3 | 4:3（1.33）| 681*512 |
| --ar 2:3 | 2:3 | 512*768 |
| --ar 3:2 | 3:2 | 768*512 |
